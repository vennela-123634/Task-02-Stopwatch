Task-02: Stopwatch Web Application
This project is a functional stopwatch web application built to measure and record time intervals.
Features
Core Functionalities: Includes start, pause, and reset functions to control the stopwatch.
Lap Tracking: Ability to track and display lap times for accurate measurement.
Tech Stack:
HTML: Used to structure the elements of the application.
CSS: Used for styling to create a user-friendly interface.
JavaScript: Used to implement the stopwatch logic and interactivity.
